3. Component Lifecycle
   - Mounting, Updating, Unmounting phases
   - Dependencies in effects
   - When to use useEffect cleanup

4. State Management
   - Context API usage and limitations

   - Prop drilling issues and solutions
   - Local state vs global state

5. Routing (React Router)
   - BrowserRouter, Routes, Route
   - Link vs NavLink
   - useNavigate hook
   - URL Parameters and query strings
   - Nested routes

6. JavaScript (ES6+) Fundamentals
   - Arrow functions and regular functions
   - Destructuring (Objects and Arrays)
   - Spread/Rest operators
   - Promises and Async/Await
   - Closures and scope
   - Higher-order functions
   - Array methods (map, filter, reduce, find, forEach)
